# SRTdash Admin Dashboard

Free admin dashboard template

## Preview

![SRTdash admin dashboard template preview](https://colorlib.com/wp/wp-content/uploads/sites/2/srtdash-html-free-admin-dashboard.jpg)

## Overview

SRTdash is a full-featured, multipurpose admin dashboard template built with Bootstrap 5.3.8, HTML5, CSS3, and vanilla JavaScript. No jQuery dependency.

It is a fully responsive template with a large collection of reusable UI components, charts, and page layouts.

## Getting Started

No build step required. Open any HTML file directly or use a local server:

```bash
# Python
cd srtdash && python3 -m http.server 8000

# Node
npx http-server srtdash

# Then visit http://localhost:8000
```

There is a [starter template](srtdash/starter.html) for creating new pages.

## Template Features

- 3 dashboard variants (ICO, Ecommerce, SEO)
- 47 total pages
- Bootstrap 5.3.8 (no jQuery)
- Vanilla JavaScript
- AVIF images with JPG/PNG fallback
- Font Awesome 7.1 + Themify Icons
- Accessible (skip links, focus-visible, reduced motion)
- Google Analytics 4 ready (placeholder ID)
- Fully responsive
- Clean, modular code
